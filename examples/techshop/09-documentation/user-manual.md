# User Manual — TechShop Online Store
**Version:** 1.0 | **Date:** 2026-03-17

---

## Getting Started

ยินดีต้อนรับสู่ TechShop ร้านขายอุปกรณ์ IT ออนไลน์ คู่มือนี้จะช่วยให้คุณเริ่มต้นใช้งานได้ภายใน 5 นาที

### สิ่งที่ต้องเตรียม

- อีเมลที่ใช้งานได้
- บัตรเครดิต/เดบิต หรือแอปธนาคารสำหรับ PromptPay
- เบราว์เซอร์ Chrome, Firefox, Safari, หรือ Edge (เวอร์ชันล่าสุด)

---

## 1. สมัครสมาชิก

1. ไปที่ [techshop.th](https://techshop.th) แล้วคลิก **สมัครสมาชิก**
2. กรอก:
   - ชื่อ-นามสกุล
   - อีเมล
   - รหัสผ่าน (อย่างน้อย 8 ตัว, มีตัวพิมพ์ใหญ่ 1 ตัว และตัวเลข 1 ตัว)
3. คลิก **สมัครสมาชิก**
4. ระบบจะพาไปหน้าหลักอัตโนมัติ

> **หมายเหตุ:** อีเมลแต่ละที่อยู่ใช้สมัครได้ครั้งเดียว

---

## 2. ค้นหาและดูสินค้า

### ค้นหาด้วย Search Bar

พิมพ์ชื่อสินค้า, แบรนด์, หรือ keyword ใน search bar ด้านบนแล้วกด Enter

### Browse ตามหมวดหมู่

คลิกเมนู **สินค้า** แล้วเลือกหมวดหมู่:
- Storage (SSD, HDD)
- RAM
- CPU
- Peripherals

### Filter และ Sort

ใช้แถบด้านซ้ายเพื่อ filter:
- **หมวดหมู่**: checkbox ✅
- **ราคา**: เลื่อน slider
- **แบรนด์**: checkbox ✅
- **Rating**: 4 ดาวขึ้นไป

Sort ผลลัพธ์ด้วย dropdown มุมขวาบน:
- ราคา: น้อย → มาก
- ราคา: มาก → น้อย
- ใหม่สุด
- Rating สูงสุด

---

## 3. เพิ่มสินค้าลงตะกร้า

1. คลิกสินค้าที่ต้องการ
2. ดูรายละเอียด (specs, รีวิว, จำนวนในสต็อก)
3. เลือกจำนวน
4. คลิก **เพิ่มลงตะกร้า**

> ไอคอนตะกร้าด้านบนขวาจะแสดงจำนวนสินค้า

---

## 4. Checkout และชำระเงิน

### ขั้นตอนที่ 1: ตรวจสอบตะกร้า

1. คลิกไอคอน 🛒 ด้านบน
2. ตรวจสอบรายการ จำนวน และราคา
3. แก้ไขจำนวนหรือลบสินค้าได้ในหน้านี้
4. คลิก **Checkout**

### ขั้นตอนที่ 2: ที่อยู่จัดส่ง

กรอกที่อยู่จัดส่ง:
- ชื่อผู้รับ
- ที่อยู่ (บ้านเลขที่, ถนน, แขวง/ตำบล)
- เมือง/จังหวัด
- รหัสไปรษณีย์
- เบอร์โทรศัพท์

### ขั้นตอนที่ 3: ชำระเงิน

**ตัวเลือก 1: บัตรเครดิต/เดบิต**
- กรอกหมายเลขบัตร, วันหมดอายุ, CVV
- ข้อมูลบัตรถูก encrypt โดย Stripe — TechShop ไม่เก็บข้อมูลบัตร

**ตัวเลือก 2: PromptPay**
- เลือก PromptPay
- ระบบแสดง QR Code พร้อมยอดเงิน
- เปิดแอปธนาคาร → สแกน QR Code
- **QR มีอายุ 15 นาที** — ถ้าหมดอายุให้กด "สร้าง QR ใหม่"

### ขั้นตอนที่ 4: ยืนยันคำสั่งซื้อ

- ตรวจสอบรายการและที่อยู่อีกครั้ง
- คลิก **ยืนยันและชำระเงิน**
- รอสักครู่ — ห้ามกดซ้ำหรือปิดหน้าต่าง
- เมื่อสำเร็จ ระบบจะแสดงหน้า **ยืนยันคำสั่งซื้อ** พร้อมเลขที่ออเดอร์

---

## 5. ติดตามคำสั่งซื้อ

1. ล็อกอิน → คลิกที่ชื่อของคุณ → **คำสั่งซื้อของฉัน**
2. เลือกออเดอร์ที่ต้องการดู
3. ดูสถานะ:

| สถานะ | ความหมาย |
|-------|---------|
| รอดำเนินการ | รับคำสั่งซื้อแล้ว กำลังตรวจสอบ |
| กำลังจัดเตรียม | กำลังแพ็กสินค้า |
| จัดส่งแล้ว | สินค้าอยู่ระหว่างส่ง + tracking number |
| ส่งถึงแล้ว | ได้รับสินค้าแล้ว |

---

## 6. คำถามที่พบบ่อย

**ลืมรหัสผ่านทำยังไง?**
คลิก "ลืมรหัสผ่าน" ที่หน้า Login → กรอกอีเมล → รับลิงก์รีเซ็ตทางอีเมล

**ยกเลิกออเดอร์ได้ไหม?**
ยกเลิกได้เฉพาะออเดอร์ที่สถานะ "รอดำเนินการ" หรือ "กำลังจัดเตรียม"
ไปที่ คำสั่งซื้อของฉัน → เลือกออเดอร์ → คลิก **ยกเลิก**

**ชำระเงินแล้วแต่ไม่ได้รับอีเมลยืนยัน**
- ตรวจสอบโฟลเดอร์ Spam/Junk
- รอสูงสุด 5 นาที
- ถ้ายังไม่ได้ ติดต่อ support@techshop.th พร้อมเลขออเดอร์

**สินค้าหมดสต็อก — จะรู้ได้ยังไงเมื่อมีสินค้า?**
คลิก **แจ้งเตือนเมื่อมีสินค้า** ที่หน้า Product Detail — ระบบจะส่งอีเมลให้
